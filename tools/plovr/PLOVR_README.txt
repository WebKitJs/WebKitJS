#
# First of all, download  a Plovr binary from 
# https://code.google.com/p/plovr/downloads/list
# and put it in this folder (/tools/plovr/) 
#
# Using Plovr:
# - replace plovr.jar with the filename of your download
#
# To start using plovr:
# in this folder, run:
java -jar plovr.jar serve plovr_config.js 

# To build, ie create a deployment version:
# in this folder, run:
# java -jar plovr.jar build plovr_config.js
#
# Learn more about Plovr at http://plovr.com/
#
# Enjoy!
